# HouseFolio

HouseFolio 是一个基于 LBS 智能分析、算法评分和 AI 决策辅助的私人找房决策管理工具。

它的目标不是提供房源，也不是替用户做交易，而是帮助租客把自己主动收集的候选房源汇集到一个私有工作台中，并通过空间数据、结构化算法和 AI 分析辅助最终决策。

## 一句话定位

HouseFolio 是一个帮助租客管理其自行收集的候选房源，并用 LBS、算法和 AI 三层决策引擎辅助比较、排序和选择的私人找房决策助手。

## 三层决策引擎

### L1 LBS：空间决策层

- 通勤时间计算
- 生活圈分析
- 地图可视化
- 多房源空间关系展示

### L2 算法：结构化决策层

- 多维度评分
- 房源排序
- 横向对比
- 相对性价比计算

### L3 AI：智能解释层

- 房源总结
- 对比分析
- 决策建议
- 风险提示解释

## 项目边界

- 不发布房源
- 不抓取房源
- 不聚合第三方平台内容
- 不撮合交易
- 不收佣金
- 不为房源真实性背书
